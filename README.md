DashTerm is an easy to use , open-source TUI dashboard which enables you to view all the essential data you need at your workplace in a single glance.

#### What's the app actually doing ? ğŸ¤”
This app helps you in marking your events throughout the day, jotting down tasks, tracking down habits, logging journals and time-boxing work with pomo sessions - all enhancing your overall productivity with lightning speed âš¡
And all this information is constantly synced with your cloud services ! 

#### Why use a TUI for this? ğŸ¤”

## How to use this âš™?
#### Requirements
- python3
- pip or pip3

#### Cloning the repo
```
>> git clone https://github.com/Akash-Sharma-1/DashTerm.git
```
#### Installing Dependencies
```
>> cd DashTerm
>> pip install -r requirements.txt
```
#### Firing up the terminal
```
>> python tui.py
```
- Use Windows Terminal for the best experience
